Game Overview
**Bee Life** is a Greenfoot game where the player controls a bee that collects flowers while avoiding raindrops.  
The goal is to reach the target score before time runs out or all lives are lost.  

Game Summary
- **Player (Bee):** Moves using arrow keys. Collects flowers for points.  
- **Flower:** Increases score when collected.  
- **Raindrop:** Decreases lives when hit.  
- **HUD:** Displays Score, Lives, and Time.
